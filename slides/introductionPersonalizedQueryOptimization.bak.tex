    Here introduce the following:
    \begin{itemize}
        \item LTQP is a client-side query approach where single engine instance services one client
        \item As with browsers and other query engines, clients exhibit patterns in what queries are issued
        \item For example, some clients might primarily query a subset of all data vaults in the ecosystem,
        thus reducing the relevant data to query.
    \end{itemize}
  }
